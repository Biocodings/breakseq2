BreakSeq2

Ultrafast and accurate nucleotide-resolution analysis of structural variants

For more information contact us at rd@bina.com

Download BreakSeq2

Latest version: https://github.com/bioinform/breakseq2/archive/2.0-alpha.tar.gz

For other versions, see "releases". https://github.com/bioinform/breakseq2/releases

System Requirements

The following Python packages must be installed:

  • pysam-0.7.7: BreakSeq2 has only been tested with version 0.7.7 of pysam.
  • Biopython
In addition, paths to the following tools must be provided as BreakSeq2 arguments:
  • BWA: Used for alignment to the breakpoint library
  • samtools: Used for alignment filtering. Only version 0.1.19 has been tested.

Installing BreakSeq2

BreakSeq2 is a python package and can be installed using pip. To install type pip install https://github.com/bioinform/breakseq2/archive/2.0-alpha.tar.gz. The current version of BreakSeq2 is 2.0-alpha. In general, the install source would be https://github.com/bioinform/breakseq2/archive/version.tar.gz

Running BreakSeq2

Type run_breakseq2.py -h for help.
